Wholesale Planning

TG JE v20234.1 introduced support for retailers planning in the wholesale space. TG JE v2024.1 extends this support to the Sales Plan, Receipt Plan, and Top Down Core Plan views.

The TG JE v2023.4 release notes extensively describes the rationale and new wholesale functionality introduced with that release. Please refer to the TG JE v2023.4 release notes for further information.

Assortment Planning

The following wholesale metrics were added to the main planning view and the Cluster Sets view for TG JE v2023.4. These metrics are now extended to the Sales Plan, Receipt Plan, and Top Down Core Plan views.

  • At Once

  • At Once Units

  • Available To Sell

  • Available To Sell Units / Total Buy Units (Receipt Plan view and TDCP view)

    Available To Sell Units (Sales Plan view)

  • Bookings Cancelled

  • Bookings Cancelled Rate (Receipt Plan view and TDCP view

  • Bookings Cancelled Units

  • Bookings Placed

    Bookings Placed Units

  • Bookings Shipped

  • Bookings Shipped Units

  • Cancel Rate (Receipt Plan view and TDCP view)

  • Sell Out Units

  • Sell Out Value

Editability

These metrics are only editable in wholesale clusters and follow the same coloring rules as existing plannable metrics (i.e. are denoted by a light blue row coloration).

  • Within TDCP, the following are editable: At Once, At Once Units, Bookings Cancelled, Bookings Cancelled Rate, and Bookings Cancelled Units.
