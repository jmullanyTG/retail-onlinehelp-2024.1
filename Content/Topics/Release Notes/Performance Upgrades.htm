Performance

  • MFP Hybrid Cost - ROW level DATA_COMPRESSION is enforced across all indexes on tbl_FinancialPlan_Common_InventoryLevelID_Cache, which entails three separate Index REBUILD operations, which will extend the upgrade script duration where many hundreds of thousands of rows are present in the table.
