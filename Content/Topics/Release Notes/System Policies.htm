System Policies

The following system policies have been added.

System Policy System Policy Group Description
Attribute Display Module: Assortment Determines which fields to show in the attributes columns. 0 - Description, 1 - Code, 2 - Code and Description
     
Config: Code/Description Display Module: Target Choice Count This policy describes how to display properties that have a code and description option, like hierarchy levels or attributes.

The options are:

  • Code

  • Description

  • Code (Description) (e.g. COH001 (Auburn))

  • Description (Code) (e.g. Auburn (COH001))

The default is Code.
