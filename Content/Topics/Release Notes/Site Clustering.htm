Site Clustering

Using financial plans as a sales attribute to cluster sites

Financial plans can now be used as a sales attribute to cluster sites with new Attribute Type Financial Plan Attribute.

Selecting Financial Plan Attribute via the Add Attribute and Edit Attribute dialogs enables users to specify the financial plan and the time range to use. The hybrid financial plans listed are those plans that are in status Initialized, In Progress, or Authorized. Once the cluster set is configured and users click Calculate on the Cluster Set window toolbar, the time range is used in conjunction with the financial plan to cluster sites.

The hybrid financial plans must have the exactly the same product hierarchy levels down to the selected clustering level. These plans can have lower levels of the product hierarchy as well but can’t be at a higher level than the required clustering. Time range selections need to be within the time range of the financial plan.
