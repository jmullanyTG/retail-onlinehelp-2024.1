Assortment Planning

Purging old or inactive curves used both in assortment planning and allocations DMDEV-87973

Functionality has been added to the existing job (Weekly) Maintenance: All Modules to garbage collect inactive and deleted curves.

The garbage collection process does the following:

  • delete references in curves table

  • delete references from profile table

  • removes curve link tags

  • removes curve from allocation strategy and marks strategy as missing configuration

  • removes curve from assortment hierarchy

  • removes curve from assortment placeholder

  • removes curve from rate of sale

  • removes curve from inventory and level forecast configuration

DMDEV-83963

Including the Attribute Lookup Description in the Manage Locations view

The Locations > Manage Locations view has been updated to include the Attribute Lookup description which is much more meaningful to users than just the Lookup Code of a Site attribute. What displays is governed by new system policy Attribute Display (Module: Assortment). See System Policies for more information.
